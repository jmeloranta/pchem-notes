\opage{
\otitle{5.10 The great and little orthogonality theorems}

\otext
Consider a group of order $h$ and let $D^{(l)}(R)$ be the representative of the operation $R$ in a $d_l$-dimensional irreducible representation of symmetry species $\Gamma^{(l)}$ of the group. The \textit{great orthogonality theorem} states that:

\aeqn{5.13}{\sum\limits_{R}D_{ij}^{(l)}(R)^*D_{i'j'}^{(l')}(R) = \frac{h}{d_l}\delta_{ll'}\delta_{ii'}\delta_{jj'}}

where * denotes complex conjugation. Here the matrices will be real and the complex conjugation will have no effect. Note that the indices $i,j$ and $i',j'$ refer to the elements within the matrix. The summation runs over all possible symmetry operations $R$ in the group.

\vspace*{0.2cm}

\textbf{Example.} Consider $C_{3v}$ with $h = 6$ and choose a) $l = \textnormal{A}_1$, $l' = \textnormal{A}_1$, $i = j = i' = j' = 1$ and b) $l = \textnormal{E}$, $l' = \textnormal{E}$, $i = i' = 3$, $j = 4$, $j' = 3$.

\vspace*{0.2cm}

\textbf{Solution.} a) For A$_1$ we have $d_l = 1$ with matrices $(1), (1), (1), (1), (1), (1)$. Here the indices $i, j, i',j'$ point to this only element in these matrices. The sum in Eq. (\ref{eq5.13}) now reads ($R = E, C_3^+, C_3^-, \sigma_v, \sigma_v', \sigma_v''$):
$$\sum\limits_RD_{11}^{(\textnormal{A}_1)}(R)^*D_{11}^{(\textnormal{A}_1)}(R) = 1\times 1 + 1\times 1 + 1\times 1 + 1\times 1 + 1\times 1 + 1\times 1 = 6$$

The right-hand-side in Eq. (\ref{eq5.13}) gives also $6/1 = 6$.

}

\opage{

\otext
b) Here we get the left-hand-side sum of Eq. (\ref{eq5.13}) to be:

$$\sum\limits_RD_{34}^{(\textnormal{E})}(R)^*D_{33}^{(\textnormal{E})}(R) = D_{34}^{(\textnormal{E})}(E)^*D_{33}^{(\textnormal{E})}(E) + D_{34}^{(\textnormal{E})}(C_3^+)^*D_{33}^{(\textnormal{E})}(C_3^+) + ...$$
$$ = 0\times 1 + 0\times 0 + 1\times 0 + 1\times 0 + 0\times 0 + 0\times 1 = 0$$

The right-hand-side of Eq. (\ref{eq5.13}) is also zero.

\vspace*{0.2cm}

The \textit{little orthogonality theorem} states that:

\aeqn{5.14}{\sum\limits_R\chi^{(l)}(R)^*\chi^{(l')}(R) = h\delta_{ll'}}

\textbf{Proof.} This result can be derived directly from the great orthogonality theorem by setting $j = i$ and $j' = i'$ and summing over the diagonal elements:

$$\sum\limits_{i,i'}\sum\limits_RD_{ii}^{(l)}(R)^*D_{i'i'}^{(l')}(R) = \sum\limits_R\left\lbrace\sum\limits_iD_{ii}^{(l)}(R)^*\right\rbrace\left\lbrace\sum\limits_{i'}D_{i'i'}^{(l')}(R)\right\rbrace$$
$$ = \sum\limits_R\chi^{(l)}(R)^*\chi^{(l')}(R)$$

}

\opage{

\otext
The right-hand-side of Eq. (\ref{eq5.13}), with the same modifications as above, becomes:

$$\sum\limits_{i,i'}\left(\frac{h}{d_l}\right)\delta_{ll'}\delta_{ii'}\delta_{ii'} = \frac{h}{d_l}\delta_{ll'}\sum\limits_i\delta_{ii} = h\delta_{ll'}$$

The little orthogonality theorem can be expressed in a simpler form by using the fact that all operations of the same class have the same character. If the number of symmetry operations in a class $c$ is $g(c)$ then:

\aeqn{5.15}{\sum\limits_c g(c)\chi^{(l)}(c)^*\chi^{(l')}(c) = h\delta_{ll'}}

For example, for $C_{3v}$ we have $g(C_3) = 2$ and $g(\sigma_v) = 3$. With $l = l'$ this becomes:

\aeqn{5.16}{\sum\limits_cg(c)\left|\chi^{(l)}(c)\right|^2 = h}

By denoting $\sqrt{g(c)}\chi_c^{(l)}$ as a component $v_c^{(l)}$ of a vector $v^{(l)}$ we can write the above equation as:

\aeqn{5.17}{\sum\limits_cv_c^{(l)*}v_c^{(l')} = \vec{v}^{(l)*} \cdot \vec{v}^{(l')} = h\delta_{ll'}}

This states that the two vectors are orthogonal unless $l = l'$. This also gives that the number of symmetry species is equal to the number of classes.

}

\opage{

\otext
The vector notation can also be applied to the great orthogonality theorem. We denote $v^{(l,i,j)} = D_{ij}^{(l)}$ and then:

\aeqn{5.18}{v^{(l,i,j)*}\cdot v^{(l',i',j')} = \frac{h}{d_l}\delta_{ll'}\delta_{ii'}\delta_{jj'}}

Since the summation in Eq. (\ref{eq5.13}) is carried out over all $h$ operations of the group, the vectors $v$ must be $h$ dimensional. The total number of vectors of a given irreducible representation is $d_l^2$ because the labels $i$ and $j$ can take $d_l$ values in a $d_l\times d_l$ matrix. The number of these orthogonal vectors clearly cannot exceed the dimension $h$. They can, in fact, be shown to be equal:

\aeqn{5.19}{\sum\limits_i d_l^2 = h}

\textbf{Example.} Derive the $C_{3v}$ character table.

\vspace*{0.2cm}

\textbf{Solution.} The order of the group is $h = 6$ and there are three different classes of symmetry operations ($E, 2C_3, 3\sigma_v$). We have already found A$_1$ and E previously. The dimensionality $d$ of the missing irrep can be found using Eq. (\ref{eq5.19}):

$$1^2 + 2^2 + d^2 = 6 \Rightarrow d = 1$$

This new irrep we will call A$_2$. With $l = \textnormal{A}_2$, Eq. (\ref{eq5.16}) gives:
\vspace*{-0.2cm}
$$\left(\chi^{(\textnormal{A}_2)}(E)\right)^2 + 2\left(\chi^{(\textnormal{A}_2)}(C_3)\right)^2 + 3\left(\chi^{(\textnormal{A}_2)}(\sigma_v)\right)^2 = h = 6$$

}

\opage{

\otext
With $l = \textnormal{A}_2$ and $l' = \textnormal{A}_1$ we get (Eq. (\ref{eq5.15})):

$$\chi^{(\textnormal{A}_2)}(E)\chi^{(\textnormal{A}_1)}(E) + 2\chi^{(\textnormal{A}_2)}(C_3)\chi^{(\textnormal{A}_1)}(C_3) + 3\chi^{(\textnormal{A}_2)}(\sigma_v)\chi^{(\textnormal{A}_1)}(\sigma_v) = 0$$

and with $l = \textnormal{A}_2$ and $l' = \textnormal{E}$:

$$\chi^{(\textnormal{A}_2)}(E)\chi^{(\textnormal{E})}(E) + 2\chi^{(\textnormal{A}_2)}(C_3)\chi^{(\textnormal{E})}(C_3) + 3\chi^{(\textnormal{A}_2)}(\sigma_v)\chi^{(\textnormal{E})}(\sigma_v) = 0$$

Substitution of the previously found characters for A$_1$ and E into the above two equations then gives:

$$\chi^{(\textnormal{A}_2)}(E) + 2\chi^{(\textnormal{A}_2)}(C_3) + 3\chi^{(\textnormal{A}_2)}(\sigma_v) = 0$$
$$2\chi^{(\textnormal{A}_2)}(E) - 2\chi^{(\textnormal{A}_2)}(C_3) = 0$$

We have three equations that can now determine three the three unknown characters for A$_2$. By solving the three equations, we get $\chi^{(\textnormal{A}_2)}(E) = 1$, $\chi^{(\textnormal{A}_2)}(C_3) = 1$, and $\chi^{(\textnormal{A}_2)}(\sigma_v) = -1$.
